# Data Analysis Labs

Лабораторные работы по predictive analytics и анализу данных на Python.

## Содержание

| Лаба | Тема | Методы |
|------|------|--------|
| `lab1_naive_bayes` | Наивный байесовский классификатор | Реализация с нуля (гауссианы), Iris, spam, starships |
| `lab2_regression` | Статистика и регрессия | Описательная статистика, очистка данных, линейная регрессия |
| `lab3_lstm` | Прогнозирование временных рядов | LSTM-сеть, TensorFlow/Keras, thermal imager |

## Запуск

```bash
pip install -r requirements.txt
python lab1_naive_bayes/NaiveBK.py
```
## Структура
```text
├── lab1_naive_bayes/   # Байесовский классификатор
├── lab2_regression/    # Статистика и регрессия
├── lab3_lstm/          # LSTM-прогнозирование
└── requirements.txt
```
